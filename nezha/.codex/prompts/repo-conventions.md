# 仓库规约加载（Codex 入口）

本仓库使用**根目录统一 Agent 文档**，无需每次指定其他路径。

**必读**：在会话或任务开始前，请读取并遵守 **根目录 `AGENTS.md`**。该文件为本仓库对所有 Agent 的统一规约（主仓与子项目、Git、BMAD、文档归属、Agent 调度等）。

请先加载根目录 **AGENTS.md**，并在后续所有行为中遵循其约定。
